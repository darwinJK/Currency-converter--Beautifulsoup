Currency value calculator
From a webpage link : "https://www.x-rates.com/", Extracting the data and converting in our console by updatinng the link based on the currency.

Using libraries like beautifulsoup4, requests.
Link : https://www.x-rates.com/calculator/?from={from_currency}&to={to_currency}&amount={amount}

from_currency - From which currecy you want to convert
to_currency - To which currency you want to convert
amount - How much amount want to convert

Major currency code :
USD - US Dollar
EUR - Euro
INR - Indian Rupee
GBP - British Pound
AUD - Australian Dollar
CAD - Canadian Dollar
SGD - Singapore Dollar

For more currecy code, visit "https://www.x-rates.com/"
